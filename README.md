# BABI's Mini Donut Promotional Website

Frontend-only promotional website built with React.js, Vite, Google Maps, and Google Gemini AI.

## Run Locally

```powershell
npm install
npm run dev
```

Then open the local URL printed by Vite, usually `http://localhost:5173`.

## Features
- **Dynamic SPA**: Fast, smooth client-side routing.
- **Responsive Grid**: Adaptive layout using CSS Grid tailored for all devices.
- **Embedded Google Maps**: Precise pin drop at the store's exact geographic coordinates.
- **AI Chatbot**: Floating smart assistant using Gemini AI, instructed specifically to talk about donuts and the shop.
- **SEO Ready**: Configured `robots.txt`, `sitemap.xml`, and Meta/Open-Graph tags for optimal Google Search visibility.

## Pages
- Home (Includes ChatBot)
- Menu
- Gallery
- About
- Location (Includes Google Maps Integration)

The About page content is based on `Business Plan.docx`, and all provided images are included under `public/assets`.
