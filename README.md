# BABI's Mini Donut Promotional Website

Frontend-only promotional website built with React.js, Vite, and Leaflet.

## Run Locally

```powershell
npm install
npm run dev
```

Then open the local URL printed by Vite, usually `http://localhost:5173`.

## Pages

- Home
- Menu
- Gallery
- About
- Location with Leaflet map

The About page content is based on `Business Plan.docx`, and all provided images are included under `public/assets/photos`.
